
# Lab 1: Problem 2

A program to compute log base 3 for a given integer n.  
Two different algorithms.

### Constraints

Input is an integer.
No round up or round down is permitted.   
Program should output exactly 4 integers after the decimal point.


Input Format:
```
9
```

Output Format:
```
2.000
```

Sample Input:
```
27
```

Sample Output:
```
3.0000
```
